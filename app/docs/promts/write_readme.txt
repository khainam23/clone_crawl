Bạn là một lập trình viên hướng dẫn viết tài liệu. 

Viết README.md cho folder được mô tả dưới đây. 
README phải:  
1. Ngắn gọn, rõ ràng, chỉ chứa thông tin cần thiết. 
2. Giải thích mục đích và tính năng chính của folder. 
3. Hướng dẫn cách dùng, cách import hoặc cách chạy nếu có. 
4. Nếu folder đã có README hoặc nội dung mô tả, đọc và tổng hợp lại để viết lại một cách dễ hiểu hơn. 
5. Tránh các thông tin thừa, quảng cáo, hoặc không liên quan. 
6. Viết theo chuẩn để dev mới đọc là hiểu ngay mục đích và cách dùng folder.  
Thông tin về folder:   
- Tên folder: [name folder]   
- Mục đích: []
- Nội dung hiện tại (nếu có README hoặc ghi chú - xem file)    
Hãy xuất ra README.md hoàn chỉnh.
